# Moveirama - Site E-commerce

Site de e-commerce para a Moveirama - loja de móveis em Curitiba e Região Metropolitana.

## Stack

- **Framework:** Next.js 16 (App Router)
- **Banco de Dados:** Supabase (PostgreSQL)
- **Hospedagem:** Vercel
- **Estilo:** Tailwind CSS + Design System Moveirama v2.0

## Desenvolvimento Local

```bash
npm install
npm run dev
```

Acesse: http://localhost:3000

## Deploy

O deploy é automático via Vercel quando houver push no GitHub.
